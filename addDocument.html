Add Document to PAPYR network

Setup MetaMask:

In order of PAPYR to work correctly you will need to install two chrome extensions: Metamask and IPFS companion.

Metamask will set up your ethereum wallet and the IPFS companion will allow you to upload the file. Once you have your metamask wallet set up, you will need to add 0.012 ETH to your MetaMask acount. You can buy ethereum by selecting the buy option on the MetaMask extension window, which you can access by clicking the fox head in the top right corner of the screen. Once you have the required amount of ethereum in your account please proceed to the next step.

Setup IPFS companion:

Once you have installed IPFS companion, you will need to configure some settings. Right click on the gray cube in the top right corner of your screen and select options. Then make sure that "Global Toggle" is checked. After that open the dropdown menu that says external and switch it to Embedded.(If you are having trouble finding this menu it is the one next to "IPFS node type")

Adding your document:

Now that you have everything set up you can now add your document. NOTE: currently the PAPYR network only supports .txt file types. More will come in the future.
